PROC SORT (NODUP / NODUPKEY) Practice Problems (Base → Advanced)

---

### 1. Basic Duplicate Removal (NODUP)

Dataset:

data emp;
input EMP_ID NAME $;
datalines;
1 Amit
2 Neha
2 Neha
3 Raj
4 John
4 John
;
run;

Task:
Remove exact duplicate rows using PROC SORT.

---

### 2. Duplicate Key Removal (NODUPKEY)

Dataset:

data emp;
input EMP_ID NAME $;
datalines;
1 Amit
2 Neha
2 Raj
3 Raj
4 John
;
run;

Task:
Keep only one record per EMP_ID using NODUPKEY.
